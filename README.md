# Getting Started

### To use these dotfiles, follow these steps:

1. Clone the repository:

```
git clone https://github.com/jajooo/dotfiles.git
```

2. Change directory to ~/dotfiles/

```
cd ~/dotfiles/
```

3. Run the install script

```
chmod +x install.sh && ./install.sh
```
